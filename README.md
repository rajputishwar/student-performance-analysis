# Student Performance Analysis

This project predicts student performance using machine learning. It analyzes how factors like study hours, attendance, and previous marks influence final results.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Data preprocessing and cleaning
- Model training using Linear Regression
- Prediction based on multiple input features
- Visualization of study patterns

## How It Works
The model is trained on a dataset containing academic inputs. It learns patterns and predicts final marks for new data.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python app.py

## Sample Output
The model predicts student marks and displays a graph showing the relationship between study hours and performance.
